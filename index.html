Import du fichier de suivi brut → extraction automatique des contrats → moteur de commissions → trésorerie agent.

Ventes brutes détectées — table agents (Google Form) rapprochée avec l'export CRM (statuts)

Un contrat MINT est reconnu par toute séquence de 5 à 7 chiffres trouvée dans la colonne « Numéro Contrat Energie » — séparateurs acceptés : espace, /, -, //, _. Chaque séquence = 1 vente MINT distincte. La colonne « Numéro Contrat ConsoPilote » (si non vide, non nulle, non « * ») déclenche une vente ConsoPilote.

Échéancier de paie — Brut MINT chaque vendredi qui suit la date de vente ; cliquez le vendredi dans le filtre ci-dessous pour isoler les lignes à régler

Détail par contrat — éditez « Statut Brut », « Statut Activation » et « Décommissionné » directement dans le tableau. Marquer « Payé » avant le vendredi de paie officiel = avance, automatiquement signalée.
